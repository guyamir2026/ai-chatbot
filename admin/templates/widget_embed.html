הטמעת widget באתר

הוסיפו את הבוט כצ'אט צף לאתר השיווקי שלכם — שורת קוד אחת בלבד.

{# הורדת ההוראות כקובץ Markdown — לבעל עסק לא-טכני שמעביר למפתח #}
לא מתעסקים בקוד?
הורידו את תוכן הדף כקובץ ושלחו למפתח האתר שלכם — הוא יידע בדיוק מה לעשות.
הורד את ההוראות למפתח
שלב 1 — הטמעה מהירה

העתיקו את השורה הבאה והדביקו אותה לפני התג </body> בכל עמוד באתר:

<script src="{{ embed_url }}" async></script>

הסקריפט יוצר כפתור צף בפינה הימנית-תחתונה של האתר. לחיצה תפתח חלון צ'אט מחובר לעוזר הדיגיטלי שלכם.

שלב 2 — התאמה אישית (אופציונלי)

אפשר לשלוט בעיצוב דרך data-* attributes:

<script src="{{ embed_url }}"
        data-position="bottom-right"
        data-color="#2563eb"
        data-font="Heebo, sans-serif"
        data-radius="20"
        data-title="העוזר של {{ widget_config.businessName }}"
        async></script>

כל ה-attributes הזמינים

{# מקור אמת יחיד: _WIDGET_DATA_ATTRIBUTES ב-admin/widget.py (משותף עם הורדת ה-Markdown, כדי שהטבלה לא תסטה) #} {% for name, default, desc in data_attributes %} {% endfor %}
שם ברירת מחדל הסבר
{{ name }}{{ default }}{{ desc }}
שלב 3 — בדיקה

נסו את ה-widget בעמוד הדגמה ייעודי:

פתיחת עמוד הדגמה

השיחה ב-widget אנונימית — לא נשמרת ב-DB ולא מופיעה בעמוד "שיחות". המבקרים מקבלים תשובות מבוססות בסיס הידע.

שלב 4 — אבטחה (מומלץ)

כדי למנוע שימוש לרעה במשאבי OpenAI שלכם, מומלץ להגביל את ה-API לאתר שלכם בלבד:

  1. בקובץ ההגדרות של השרת (Render → Environment), הוסיפו משתנה:
WIDGET_ALLOWED_ORIGINS=https://your-site.com,https://www.your-site.com

ללא הגדרה — ה-widget פועל מכל אתר. עם רשימה — בקשות מ-origin אחר ייחסמו (HTTP 403).

יש גם תקרת בקשות פר-IP (WIDGET_RATE_LIMIT_PER_HOUR, ברירת מחדל 30) כקו הגנה שני.
